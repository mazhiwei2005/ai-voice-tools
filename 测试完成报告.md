## 🚀 快速开始

### 方案一：GPT-SoVITS（推荐）

1. **解压集成包**
   - 使用 7-Zip 或 WinRAR 解压 `GPT-SoVITS-v3lora-20250228.7z`
   - 解压到 `D:\stdio\GPT-SoVITS-Complete\`

2. **启动**
   - 双击 `go-webui.bat`
   - 浏览器自动打开 http://127.0.0.1:9874

3. **使用**
   - 上传 5-10 秒目标角色音频
   - 输入文字
   - 点击生成
   - 获得角色语音！

### 方案二：RVC（手动安装）

1. **安装依赖**
   - 双击 `安装RVC依赖.bat`
   - 或手动执行：
     ```batch
     cd D:\stdio\RVC
     pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu117
     pip install -r requirements.txt
     ```

2. **启动**
   - 双击 `启动RVC.bat`
   - 或运行：`python infer-web.py`
   - 浏览器打开 http://127.0.0.1:7897

3. **下载模型**
   - 双击 `下载热门模型.bat`
   - 或从 Hugging Face 下载

## 💡 使用技巧

### GPT-SoVITS（文字转语音）
1. 准备 5-10 秒目标角色清晰音频
2. 在 WebUI 中上传音频
3. 输入文字
4. 点击生成
5. 获得角色语音！

### RVC（声音转换）
1. 下载预训练模型
2. 上传你的音频
3. 选择目标声音
4. 点击转换
5. 获得转换后的语音！

### 组合使用（推荐）
1. 用 GPT-SoVITS 生成基础语音
2. 用 RVC 进一步调整声音风格
3. 获得最佳效果！

## ⚠️ 注意事项

### 显存要求
- **GPT-SoVITS**: 建议 4GB+ 显存
- **RVC**: 建议 2GB+ 显存
- **你的 RTX 3070**: 8GB 显存，完全够用！

### 音频质量
- 输入音频越清晰，效果越好
- 建议使用无背景音乐的纯人声
- 音频长度：5-10 秒最佳

### 训练自己的模型
- 需要 5-10 分钟目标角色音频
- 训练时间约 1-2 小时
- 效果比预训练模型更好

## 🆘 常见问题

### Q: 解压失败？
A: 请安装 7-Zip 或 WinRAR

### Q: 显存不足？
A: 在设置中降低 batch_size

### Q: 生成的声音有杂音？
A: 确保输入音频清晰，调整 denoise 参数

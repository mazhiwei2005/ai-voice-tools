# 🎭 AI 语音克隆工具包

## 📦 工具列表

| 工具 | 功能 | 启动方式 |
|------|------|----------|
| **GPT-SoVITS** | 文字 → 语音（TTS） | 双击 `启动GPT-SoVITS.bat` |
| **RVC** | 语音 → 语音（变声） | 双击 `启动RVC.bat` |

## 🚀 快速开始

### 第一步：启动工具
- 双击 `启动GPT-SoVITS.bat` 或 `启动RVC.bat`
- 首次运行会自动安装依赖（需要联网）

### 第二步：下载模型
- 双击 `下载模型.bat` 查看可用模型
- 从 Hugging Face 下载预训练模型
- 或者自己训练（需要 5-10 分钟音频）

### 第三步：开始使用
- **GPT-SoVITS**: 上传音频 → 输入文字 → 生成语音
- **RVC**: 选择模型 → 上传音频 → 转换声音

## 🎯 使用场景

### 场景1：视频配音
1. 写好台词
2. 用 GPT-SoVITS 生成语音
3. 导入剪辑软件同步

### 场景2：实时变声
1. 启动 RVC
2. 连接麦克风
3. 实时转换声音（适合直播）

### 场景3：唱歌转换
1. 录制你的歌声
2. 用 RVC 转换成目标角色
3. 生成动漫角色翻唱

## 💡 小贴士

1. **音频质量很重要**：输入音频越清晰，效果越好
2. **训练时间**：自己训练模型约需 1-2 小时
3. **显存要求**：建议 4GB 以上显存，没有可用 CPU 模式
4. **模型分享**：B站有很多UP主分享训练好的模型

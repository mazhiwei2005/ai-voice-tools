# 🎬 视频处理工具 - 拖拽功能使用说明

## ✅ 功能已添加

我已经为视频处理工具添加了拖拽功能！

### 🎯 新功能

1. **拖拽文件夹到正片区**
   - 直接拖拽文件夹到正片区的列表
   - 自动识别文件夹内的视频文件
   - 自动添加到批量文件夹合并列表

2. **拖拽视频文件到正片区**
   - 直接拖拽视频文件到正片区的列表
   - 正常添加到视频列表

3. **智能识别**
   - 自动区分文件夹和视频文件
   - 自动检查文件夹内是否有视频
   - 避免重复添加

---

## 🚀 使用方法

### 方法一：拖拽文件夹（推荐）

1. 打开文件管理器
2. 选中多个文件夹
3. 拖拽到视频处理工具的正片区列表
4. 工具会自动识别并添加到批量合并列表

**效果：**
```
📁 文件夹1 (10个视频)
📁 文件夹2 (8个视频)
📁 文件夹3 (12个视频)
```

### 方法二：拖拽视频文件

1. 打开文件管理器
2. 选中多个视频文件
3. 拖拽到视频处理工具的正片区列表
4. 工具会自动添加到视频列表

**效果：**
```
video1.mp4
video2.mp4
video3.mp4
```

### 方法三：混合拖拽

1. 同时选中文件夹和视频文件
2. 拖拽到正片区列表
3. 工具会自动分类处理

---

## 📋 操作步骤

### 步骤一：启动工具

```batch
cd D:\stdio\视频处理工具
python video_processor.py
```

或双击 `启动.bat`

### 步骤二：拖拽文件

1. 打开文件管理器
2. 导航到 H:\pjsp 或其他位置
3. 选中要处理的文件夹
4. 拖拽到正片区的列表区域

### 步骤三：开始处理

1. 确认列表中的文件夹和视频
2. 点击"开始处理"按钮
3. 工具会自动逐个合并文件夹内的视频

---

## 💡 使用技巧

### 批量处理多个文件夹

1. **选中多个文件夹**
   - 按住 Ctrl 键点击多个文件夹
   - 或按住 Shift 键选择范围

2. **拖拽到正片区**
   - 直接拖拽到列表区域
   - 工具会自动识别并添加

3. **开始处理**
   - 点击"开始处理"
   - 工具会自动逐个合并

### 混合添加

1. **同时拖拽文件夹和视频**
   - 文件夹会添加到批量合并列表
   - 视频文件会添加到普通视频列表

2. **分别处理**
   - 文件夹会自动合并
   - 视频文件会单独处理

## ⚠️ 注意事项

### 文件夹要求

- 文件夹内必须有视频文件（.mp4, .mkv, .avi, .flv, .ts）
- 空文件夹会被跳过
- 已添加的文件夹不会重复添加

### 视频文件要求

- 支持格式：.mp4, .mkv, .avi, .flv, .ts
- 文件必须存在且可读
- 已添加的视频不会重复添加

### 拖拽操作

- 支持同时拖拽多个项目
- 支持混合拖拽文件夹和视频
- 拖拽时会显示添加日志

## 🔧 故障排除

### Q: 拖拽没有反应？

A:
1. 检查是否安装了 tkinterdnd2
2. 检查工具是否显示"拖拽功能已启用"
3. 尝试重启工具

### Q: 文件夹没有被识别？

A:
1. 检查文件夹内是否有视频文件
2. 检查视频格式是否支持
3. 查看日志中的警告信息

### Q: 重复添加了文件夹？

A:
1. 工具会自动检查重复
2. 如果还是重复，可以手动移除
3. 点击"清空"按钮清除所有

### Q: 想重新添加某个文件夹？

A:
1. 先移除已添加的文件夹
2. 重新拖拽添加
3. 或使用"导入文件夹"按钮


**结果：**
- 文件夹添加到批量合并列表
- 视频添加到普通视频列表
- 分别处理

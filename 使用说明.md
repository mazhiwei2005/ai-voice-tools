# 🎤 AI 语音工具使用指南

## 工具说明

### 1. GPT-SoVITS（推荐新手）
- **功能**：文字 → 语音（直接把文字转成角色声音）
- **特点**：只需 5 秒音频即可克隆声音，支持中日英
- **启动**：双击 `启动GPT-SoVITS.bat`
- **访问**：http://127.0.0.1:9874

### 2. RVC（声音转换）
- **功能**：语音 → 语音（把你的声音变成目标角色）
- **特点**：实时变声，社区模型多
- **启动**：双击 `启动RVC.bat`
- **访问**：http://127.0.0.1:7865

---

## 快速上手

### 方法一：GPT-SoVITS（文字转语音）

1. 双击 `启动GPT-SoVITS.bat`
2. 等待自动安装依赖（首次运行较慢）
3. 浏览器自动打开 WebUI
4. 上传 5-10 秒目标角色的音频
5. 输入文字，点击生成

### 方法二：RVC（声音转换）

1. 双击 `启动RVC.bat`
2. 等待自动安装依赖
3. 浏览器自动打开 WebUI
4. 下载或导入预训练模型（见下方模型下载）
5. 上传你的音频，选择目标声音，点击转换

## 使用技巧

### GPT-SoVITS 训练自己的模型
1. 收集目标角色 5-10 分钟清晰音频
2. 去除背景音乐和杂音
3. 在 WebUI 中上传并标注
4. 开始训练（约 1-2 小时）
5. 训练完成后即可使用

### RVC 模型使用
1. 下载 .pth 模型文件
2. 下载对应的 .index 文件
3. 放入 RVC 的 models 文件夹
4. 在 WebUI 中选择模型
5. 上传音频，调整参数，点击转换

## 常见问题

### Q: 首次启动很慢？
A: 需要下载 PyTorch 等大型依赖，耐心等待

### Q: 显存不足？
A: 在设置中降低 batch_size，或使用 CPU 模式

### Q: 生成的声音有杂音？
A: 确保输入音频清晰，调整 denoise 参数

### Q: 想做视频配音？
A: 推荐组合：GPT-SoVITS（文字转语音）+ 剪辑软件

## 进阶用法

### 批量配音脚本
可以写 Python 脚本调用 API 批量处理：
- GPT-SoVITS API: http://127.0.0.1:9880
- RVC API: http://127.0.0.1:7865/api

### 实时变声
RVC 支持实时麦克风输入，可用于直播

### 视频制作流程
1. 写好台词文字
2. 用 GPT-SoVITS 生成语音
3. 导入剪辑软件（PR/剪映）
4. 同步画面和音频
5. 导出视频

## 🔧 RVC 安装

### 方案一：使用安装脚本
双击 `安装RVC依赖.bat`

### 方案二：手动安装
```batch
cd D:\stdio\RVC

:: 安装 PyTorch (CUDA 11.7)
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu117

:: 安装依赖
pip install -r requirements.txt

:: 安装额外依赖
pip install faiss-gpu

:: 启动
python infer-web.py
```

### 方案三：使用 Conda（推荐）
```bash
conda create -n rvc python=3.10
conda activate rvc
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu117
pip install -r requirements.txt
python infer-web.py
```

## 🎯 快速开始流程

### 流程一：GPT-SoVITS（文字转语音）
```
1. 下载并解压 GPT-SoVITS 集成包
2. 双击 go-webui.bat 启动
3. 上传 5-10 秒目标角色音频
4. 输入文字
5. 点击生成
6. 获得角色语音！
```

### 流程二：RVC（声音转换）
```
1. 安装 RVC 依赖
2. 启动 RVC WebUI
3. 下载预训练模型
4. 上传你的音频
5. 选择目标声音
6. 点击转换
7. 获得转换后的语音！
```

### 流程三：组合使用（推荐）
```
1. 用 GPT-SoVITS 生成基础语音
2. 用 RVC 进一步调整声音风格
3. 获得最佳效果！
```

## ⚠️ 注意事项

### 显存要求
- **GPT-SoVITS**: 建议 4GB+ 显存
- **RVC**: 建议 2GB+ 显存
- **你的 RTX 3070**: 8GB 显存，完全够用！

### 音频质量
- 输入音频越清晰，效果越好
- 建议使用无背景音乐的纯人声
- 音频长度：5-10 秒最佳

### 训练自己的模型
- 需要 5-10 分钟目标角色音频
- 训练时间约 1-2 小时
- 效果比预训练模型更好

## 🆘 常见问题

### Q: 下载很慢？
A: 使用国内镜像或下载工具

### Q: 显存不足？
A: 在设置中降低 batch_size

### Q: 生成的声音有杂音？
A: 确保输入音频清晰，调整 denoise 参数

### Q: 想做视频配音？
A: 推荐：GPT-SoVITS + 剪辑软件

### Q: 如何实时变声？
A: 使用 RVC 的实时模式，连接麦克风

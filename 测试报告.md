## 🚀 快速开始

### 方法一：GPT-SoVITS（推荐）

1. **等待下载完成**
   - 下载 `GPT-SoVITS-v3lora-20250228.7z` (约 3-4 GB)

2. **解压文件**
   - 使用 7-Zip 或 WinRAR 解压
   - 解压到 `D:\stdio\GPT-SoVITS-Complete\`

3. **启动**
   - 进入解压后的文件夹
   - 双击 `go-webui.bat`
   - 浏览器自动打开 http://127.0.0.1:9874

4. **使用**
   - 上传 5-10 秒目标角色音频
   - 输入文字
   - 点击生成

### 方法二：RVC（手动安装）

1. **安装依赖**
   - 双击 `安装RVC依赖.bat`
   - 或手动执行：
     ```batch
     cd D:\stdio\RVC
     pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu117
     pip install -r requirements.txt
     ```

2. **启动**
   - 双击 `启动RVC.bat`
   - 或运行：`python infer-web.py`
   - 浏览器打开 http://127.0.0.1:7897

3. **下载模型**
   - 双击 `下载模型.bat`
   - 或从 Hugging Face 下载

## ⚠️ 注意事项

1. **显存要求**
   - GPT-SoVITS: 建议 4GB+ 显存
   - RVC: 建议 2GB+ 显存
   - 你的 RTX 3070 (8GB) 完全够用

2. **首次运行**
   - 会自动下载模型（约 1-2 GB）
   - 请耐心等待

3. **音频质量**
   - 输入音频越清晰，效果越好
   - 建议使用无背景音乐的纯人声

4. **训练自己的模型**
   - 需要 5-10 分钟目标角色音频
   - 训练时间约 1-2 小时

## 🔧 故障排除

### Q: 下载很慢？
A: 使用镜像源或下载工具

### Q: 显存不足？
A: 在设置中降低 batch_size

### Q: 生成的声音有杂音？
A: 确保输入音频清晰，调整 denoise 参数

### Q: 想做视频配音？
A: 推荐：GPT-SoVITS + 剪辑软件
